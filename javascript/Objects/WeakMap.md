## Instance Methods (Prototype)

| Syntax | Description | Example | Output |
|---|---|---|---|
| `.set()` | Adds or updates a key-value pair (key must be object) | `wm.set(obj, 123)` | `WeakMap {...}` |
| `.get()` | Returns the value of a key | `wm.get(obj)` | `123` |
| `.has()` | Checks if a key exists | `wm.has(obj)` | `true` |
| `.delete()` | Removes a key-value pair | `wm.delete(obj)` | `true` |
