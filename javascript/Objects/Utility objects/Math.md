# JavaScript Math

## Static Methods

| Syntax | Description | Example | Output |
|---|---|---|---|
| `Math.random()` | Returns a random number between 0 and 1 | `Math.random()` | `0.5342...` |
| `Math.floor()` | Rounds a number down | `Math.floor(4.9)` | `4` |
| `Math.ceil()` | Rounds a number up | `Math.ceil(4.1)` | `5` |
| `Math.round()` | Rounds to nearest integer | `Math.round(4.5)` | `5` |
| `Math.abs()` | Returns absolute value | `Math.abs(-10)` | `10` |
| `Math.max()` | Returns the largest number | `Math.max(1, 5, 3)` | `5` |
| `Math.min()` | Returns the smallest number | `Math.min(1, 5, 3)` | `1` |
| `Math.pow()` | Raises a number to a power | `Math.pow(2, 3)` | `8` |
| `Math.sqrt()` | Returns square root | `Math.sqrt(16)` | `4` |
| `Math.trunc()` | Removes decimal part | `Math.trunc(4.9)` | `4` |


## Static Properties

| Property | Description | Example | Output |
|---|---|---|---|
| `Math.PI` | Value of π | `Math.PI` | `3.141592653589793` |
| `Math.E` | Euler’s number | `Math.E` | `2.718281828459045` |
| `Math.SQRT2` | Square root of 2 | `Math.SQRT2` | `1.4142135623730951` |
| `Math.LN2` | Natural log of 2 | `Math.LN2` | `0.6931471805599453` |
